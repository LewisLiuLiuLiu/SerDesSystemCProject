# IEEE DVCon China 2026 论文格式检查清单

## 📄 页面设置

### 纸张大小
- [ ] **纸张类型**: US Letter (8.5 × 11 英寸)
- [ ] **页面方向**: 纵向

### 页边距
- [ ] **上边距**: 1 英寸 (25.4 mm)
- [ ] **下边距**: 1 英寸 (25.4 mm)
- [ ] **左边距**: 1 英寸 (25.4 mm)
- [ ] **右边距**: 1 英寸 (25.4 mm)
- [ ] **页眉距离**: 960 twips
- [ ] **页脚距离**: 960 twips

### 页面布局
- [ ] **列数**: 单列 (Single Column)
- [ ] **列宽**: 8640 twips (约 6 英寸)
- [ ] **行距**: 单倍行距 (Single-spaced)
- [ ] **页码**: 不添加页码 (Do not add page numbers)

## 🔤 字体要求

### 默认字体
- [ ] **主要字体**: Times New Roman
- [ ] **中文字体**: 等线

### 字体大小规格 (Table I)

| 字体大小 (pt) | 用途 | 字体样式 |
|--------------|------|---------|
| **6** | 表格标题、表格上标 | 常规 |
| **8** | 章节标题、参考文献、表格、表格名称、图注首字母、脚注、上下标 | 常规 |
| **9** | 作者姓名 | 常规 |
| **10** | 作者单位、正文、公式、章节标题首字母 | 常规 (子标题用斜体) |
| **11** | 作者姓名 | 常规 |
| **24** | 论文标题 | 常规 |

### 字体样式要求
- [ ] **Abstract 标签**: 斜体
- [ ] **Type sizes and typefaces**: 斜体
- [ ] **Subheading**: 斜体 (10 pt)
- [ ] **上标**: 使用正确的上标格式 (如 10³)
- [ ] **下标**: 使用正确的下标格式 (如 μ₀)
- [ ] **罗马符号**: 斜体表示变量和数量
- [ ] **希腊符号**: 不斜体
- [ ] **单位符号**: 不斜体

## 📝 段落格式

### 正文段落
- [ ] **行距**: 240 twips (单倍行距)
- [ ] **对齐**: 左右对齐 (Left- and right-justify)
- [ ] **段落缩进**: 3.5 mm (0.14 英寸)
- [ ] **段后空格**: 使用自动断字
- [ ] **拼写检查**: 检查拼写

### 标题格式

#### 一级标题 (Heading 1)
- [ ] **字体大小**: 8 pt
- [ ] **字体样式**: 常规 (全部大写)
- [ ] **编号格式**: 罗马数字 + 句号 + 3个空格 (如 "I.   Introduction")
- [ ] **行距**: 240 twips
- [ ] **段前间距**: 0
- [ ] **段后间距**: 0

#### 二级标题 (Heading 2)
- [ ] **字体大小**: 10 pt
- [ ] **字体样式**: 常规
- [ ] **编号格式**: 字母 + 句点 + Tab (如 "A.    Full-Sized Copy")
- [ ] **行距**: 240 twips
- [ ] **段前间距**: 0

#### Abstract
- [ ] **字体大小**: 9 pt
- [ ] **字体样式**: "Abstract" 标签斜体，正文常规
- [ ] **行距**: 240 twips
- [ ] **段后间距**: 0

### 列表格式
- [ ] **编号列表**: 使用阿拉伯数字加括号 (如 "1)", "2)")
- [ ] **列表缩进**: 正确缩进
- [ ] **行距**: 240 twips

## 📊 表格格式

### 表格位置
- [ ] **表格标题**: 居中在表格上方
- [ ] **位置**: 尽量放在页面顶部或底部
- [ ] **避免**: 避免放在列中间
- [ ] **引用**: 避免在文中首次提及之前放置表格

### 表格标题格式
- [ ] **表格编号**: "TABLE I", "TABLE II" 等 (全部大写)
- [ ] **字体大小**: 8 pt
- [ ] **对齐**: 居中
- [ ] **段前间距**: 0

### 表格内容格式
- [ ] **边框**: 单线边框 (sz="4")
- [ ] **字体大小**: 8 pt (表格内)
- [ ] **对齐**: 内容居中
- [ ] **表头**: 加粗 (Bold)
- [ ] **行高**: 合适的行高设置

### 表格脚注
- [ ] **使用字母**: 使用字母而非数字作为表格脚注
- [ ] **位置**: 表格底部
- [ ] **格式**: 上标字母

---

## 🖼️ 图片格式

### 图片位置
- [ ] **图片标题**: 居中在图片下方
- [ ] **位置**: 尽量放在页面顶部或底部
- [ ] **避免**: 避免放在列中间
- [ ] **引用**: 避免在文中首次提及之前放置图片

### 图片标题格式
- [ ] **格式**: "Figure 1. 图像描述" (首字母大写，句子结构)
- [ ] **字体大小**: 8 pt
- [ ] **对齐**: 居中
- [ ] **行距**: 240 twips

### 图表标签要求
- [ ] **轴标签**: 使用文字而非符号
  - ✅ 正确: "Magnetization" 或 "Magnetization, M"
  - ❌ 错误: "M"
- [ ] **单位**: 将单位放在括号内
  - ✅ 正确: "Magnetization (A/m)" 或 "Magnetization (A · m⁻¹)"
  - ❌ 错误: "Magnetization/K"
- [ ] **不使用比值**: 不要仅用单位比率标注轴
  - ❌ 错误: "Temperature/K"
  - ✅ 正确: "Temperature (K)"
- [ ] **乘数**: 清楚说明乘数
  - ✅ 正确: "Magnetization (kA/m)" 或 "Magnetization (10³ A/m)"
- [ ] **标签大小**: 图表标签应清晰可读，约 10 pt
- [ ] **标签居中**: 标题居中在列内

## 🔬 公式格式

### 公式编号
- [ ] **编号格式**: 连续编号，编号放在圆括号内，靠右对齐
  - 示例: `a + b = c.  (1)`
- [ ] **位置**: 公式编号靠右边缘对齐

### 公式样式
- [ ] **符号**: 罗马符号斜体，希腊符号不斜体
- [ ] **负号**: 使用 en dash (–) 而非连字符 (-)
- [ ] **除法**: 使用斜杠 (/)、exp 函数或指数以使公式更紧凑
- [ ] **括号**: 使用括号避免分母中的歧义
- [ ] **标点**: 公式作为句子的一部分时，用逗号或句号标点

### 公式引用
- [ ] **格式**: 使用 "(1)" 而非 "Eq. (1)" 或 "equation (1)"
- [ ] **句子开头**: 仅在句子开头使用 "Equation (1)"
  - ✅ 正确: "Equation (1) is ..."
  - ✅ 正确: "... as in (1)."
- [ ] **符号定义**: 在公式出现之前或紧随其后定义符号

## 📚 参考文献

### 引用格式
- [ ] **编号**: 方括号内的连续编号 [1], [2], [3]
- [ ] **位置**: 标点符号跟在括号后面
  - ✅ 正确: "... as shown in [1]."
  - ✅ 正确: "... method [2], which..."
- [ ] **句子开头**: 使用 "Ref. [3]" 或 "Reference [3]"
  - ✅ 正确: "Reference [3] was the first..."

### 参考文献列表格式

#### 期刊论文
```
[1]  G. Eason, B. Noble, and I.N. Sneddon, "On certain integrals of Lipschitz-Hankel
     type involving products of Bessel functions," Phil. Trans. Roy. Soc. London,
     vol. A247, pp. 529-551, April 1955.
```

#### 书籍
```
[2]  J. Clerk Maxwell, A Treatise on Electricity and Magnetism, 3rd ed., vol. 2.
     Oxford: Clarendon, 1892, pp. 68-73.
```

#### 书籍章节
```
[3]  I.S. Jacobs and C.P. Bean, "Fine particles, thin films and exchange anisotropy,"
     in Magnetism, vol. III, G.T. Rado and H. Suhl, Eds. New York: Academic, 1963,
     pp. 271-350.
```

#### 未发表论文
```
[4]  K. Elissa, "Title of paper if known," unpublished.
```

#### 已接收待发表
```
[5]  R. Nicole, "Title of paper with only first word capitalized," J. Name Stand. Abbrev.,
     in press.
```

#### 译文期刊
```
[6]  Y. Yorozu, M. Hirano, K. Oka, and Y. Tagawa, "Electron spectroscopy studies on
     magneto-optical media and plastic substrate interface," IEEE Transl. J. Magn. Japan,
     vol. 2, pp. 740-741, August 1987 [Digests 9th Annual Conf. Magnetics Japan,
     p. 301, 1982].
```

#### 其他
```
[7]  M. Young, The Technical Writer's Handbook. Mill Valley, CA: University Science, 1989.
```

### 参考文献要求
- [ ] **作者名单**: 列出所有作者，6人或以上使用 "et al."
- [ ] **期刊前缀**: IEEE Transactions 不再使用期刊前缀
  - ✅ 正确: "IEEE Trans. Magn., vol. 25,"
  - ❌ 错误: "vol. MAG-25."
- [ ] **标题大小写**: 论文标题中首字母大写，其他单词大写，除非是连词、少于7个字母的介词和介词短语
- [ ] **未发表**: 已投稿但未发表的论文标注为 "unpublished"
- [ ] **已接收**: 已接收待发表的论文标注为 "in press"
- [ ] **译文**: 译文期刊先给出英文引用，再给出原文引用
- [ ] **期刊名**: 期刊名斜体

### 脚注
- [ ] **编号**: 单独编号，使用上标
- [ ] **位置**: 放在引用该脚注的列的底部
- [ ] **不在参考文献**: 不将脚注放在参考文献列表中
- [ ] **表格脚注**: 使用字母而非数字

## 📐 单位系统

### 单位选择
- [ ] **主要单位**: 使用 SI (MKS) 或 CGS 作为主要单位 (推荐 SI)
- [ ] **次要单位**: 英制单位可作为次要单位放在括号内
- [ ] **例外**: 英制单位作为贸易标识符时可单独使用 (如 "3.5-inch disk drive")

### 单位格式
- [ ] **避免混合**: 避免混合 SI 和 CGS 单位
- [ ] **方程平衡**: 如必须混合使用，明确说明方程中每个量的单位
- [ ] **小数点**: 小数点前加零
  - ✅ 正确: "0.25"
  - ❌ 错误: ".25"
- [ ] **立方厘米**: 使用 "cm³" 而非 "cc"
- [ ] **单位一致性**: 不要混合完整拼写和缩写
  - ✅ 正确: "Wb/m²" 或 "webers per square meter"
  - ❌ 错误: "webers/m²"
- [ ] **文中单位**: 单位在文本中应完整拼写
  - ✅ 正确: "...a few henries,"
  - ❌ 错误: "...a few H."
- [ ] **上下标**: 正确使用上下标
  - ✅ 正确: "cm³"
  - ❌ 错误: "cm3"

## 📝 缩写词和首字母缩略词

### 定义规则
- [ ] **首次使用**: 首次在文本中使用时定义缩写词和首字母缩略词
- [ ] **Abstract 也要定义**: 即使在 abstract 中已定义，也要在正文中再次定义
- [ ] **无需定义**: 以下缩写无需定义
  - IEEE, SI, MKS, CGS, sc, dc, rms
- [ ] **标题中**: 除非不可避免，否则不要在标题中使用缩写

## ✏️ 标点和语法

### 标点符号
- [ ] **句号后空格**: 使用两个空格
  - ✅ 正确: "word1.  word2"
- [ ] **引号内标点**: 美式英语中，句号和逗号在引号内
  - ✅ 正确: "like this period."
- [ ] **括号标点**: 句末的括号语句标点在右括号外
  - ✅ 正确: "... (like this)."
- [ ] **完整句子**: 括号内的完整句子标点在括号内
  - ✅ 正确: "(A parenthetical sentence is punctuated within the parentheses.)"
- [ ] **et al.**: "et al." 中的 "et" 后没有句号
- [ ] **i.e. 和 e.g.**: 正确使用
  - "i.e." = "that is"
  - "e.g." = "for example"

### 连字符
- [ ] **复杂修饰词**: 复合修饰词使用连字符
  - ✅ 正确: "zero-field-cooled magnetization"

### 避免的错误
- [ ] **悬垂分词**: 避免悬垂分词
  - ❌ 错误: "Using (1), the potential was calculated."
  - ✅ 正确: "The potential was calculated using (1),"
  - ✅ 正确: "Using (1), we calculated the potential."

## 🎯 语言和写作

### 词汇选择
- [ ] **data**: "data" 是复数，不是单数
- [ ] **μ₀**: 真空磁导率的下标是零，不是小写字母 "o"
- [ ] **inset**: 图中的图称为 "inset"，不是 "insert"
- [ ] **alternatively**: 优先使用 "alternatively" 而非 "alternately" (除非表示交替)
- [ ] **essentially**: 不要用 "essentially" 表示 "approximately" 或 "effectively"

### 同音词区分
- [ ] **affect vs effect**: 正确使用 "affect" (动词) 和 "effect" (名词)
- [ ] **complement vs compliment**: 正确使用
- [ ] **discreet vs discrete**: 正确使用
- [ ] **principal vs principle**: 正确使用
- [ ] **imply vs infer**: 不要混淆

### 前缀
- [ ] **non-**: "non" 不是一个单词，应连接到它修饰的词上，通常不加连字符

### 大小写规则
- [ ] **Acknowledgment**: 美式拼写 "acknowledgment" 中 "g" 后没有 "e"
- [ ] **标题大写**: 使用标题大小写格式
- [ ] **章节标题**: 章节标题全大写
- [ ] **子标题**: 子标题首字母大写，其他常规
